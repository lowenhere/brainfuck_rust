Brainfuck interpreter written in Rust

## Build 
``` bash
# build
cargo build --release
```

This will build a binary at `target/release/brainfuck_rust`.

## Usage
Some example programs are included in `examples/`.

```bash
cat examples/hello_world.bf | brainfuck_rust
```
